Репозиторий для лабораторных работ
## 2 Лабораторная работа
1. Получить файлик HelloWorld.java с ветки мастер 
1.1. git checkout master
1.2. git pull     или git pull origin master
1.3. Вернуться на свою ветку
1.4. git checkout HelloWorld.java master  или git merge master
2. Изменить содержимое файла HelloWorld так, чтобы на экране 
отображалась ваша фамили имя отчество
3. Скомпилировать программу javac HelloWorld.java
4. Выполнить программу java HelloWorld
